# 4K Movie Kesri - Studio Management Platform

Professional Photography & Videography studio management suite designed for high-fidelity, cinematic user experiences.

## Vision
To provide a seamless, premium administrative interface for managing one of Surat's premier photography studios, facilitating everything from client inquiries to AI-powered content generation.

## Technical Stack
- **Framework**: Next.js 16 (App Router)
- **Frontend**: React 19, Tailwind CSS 4, Framer Motion (Animations)
- **State Management**: Zustand
- **Backend & Database**: Firebase (Authentication, Firestore, Storage)
- **AI Engine**: Groq (Llama 3.3 70B)
- **Media**: Cloudinary Integration
- **Communication**: Nodemailer (Inquiry notifications)

## Core Modules
- **Admin Dashboard**: Central command center for studio operations.
- **Blog Manager**: AI-assisted content creation for SEO.
- **Gallery Manager**: Premium lightbox and border-framed media organization.
- **Equipment Tracker**: Logistics management for photography gear.
- **Staff Manager**: Personnel and assignment tracking.
- **Inquiry System**: Client lead management.

## Environment Context
- **OS**: Windows
- **Shell**: PowerShell
- **Agent**: Antigravity (Gemini 3 Pro)
